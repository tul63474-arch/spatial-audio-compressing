# spatial-audio-compressingA Streamlit-based web application for compressing and analyzing 1st-order Ambisonic (B-format) audio using Principal Component Analysis (PCA).

# Features:
# Audio Compression: Interactive PCA-based compression for spatial audio.

# Spatial Metrics: Real-time calculation of SNR, Latency, and Energy Vectors.

# Visual Analysis:Time-domain waveforms and spectrogram error analysis.

# Directional Masking and Perceptual thresholds.

# Binaural Cue Distribution (ITD/ILD).

# Polar plots for Spatial Localization (Azimuth & Elevation).

# Playback: Integrated binaural rendering for immediate listening of reconstructed audio.
